# Todo List App with TypeScript and React

The Todo List App is a simple and powerful task management application built using React and TypeScript. This app helps you stay organized by allowing you to create, track, complete, and remove tasks in a user-friendly interface.

![App Screenshot](screenshot.png)

## Features

- Add tasks to your list and keep track of your to-dos.
- Mark tasks as completed with a single click.
- Revert completed tasks back to their original state.
- Remove tasks that are no longer needed.
- Responsive design for seamless usage on various devices.

## Demo

Check out the [Todo List App](https://a-basuony.github.io/To_Do_app-with-TypeScript-React/) live on GitHub Pages.

## Getting Started

Follow these steps to get started with the Todo List App:

1. Clone this repository to your local machine.
2. Navigate to the project directory: `cd To_Do_app-with-TypeScript-React`.
3. Install dependencies: `npm install`.
4. Start the development server: `npm start`.

The app will be available at `http://localhost:3000` in your browser.
